# To-Do List Group Project
This project is a simple and interactive To-Do List Application developed as part of a group assignment.

## How to Run the Project
1. Open `SettingUpProject.html` in a browser.
2. Follow the instructions on the page to add, mark, and remove tasks.

## Directory Structure
- Task1_SettingUpProject: Contains the HTML and basic JS setup.
- Task2_DOM_Manipulation_and_Styling: Contains dynamic task management logic and CSS for styling.
- Task3_Error_Handling: Contains error-handling examples.
- Task4_User_Interaction: Contains user interaction logic like marking tasks as completed.
- Task5_Final_Touches: Final improvements and testing logic.

## Features
- Add, remove, and mark tasks as completed.
- Clear all completed tasks.
- Error handling and debugging.
